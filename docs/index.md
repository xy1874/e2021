# 实验文档简介

本实验文档为哈尔滨工业大学（深圳）《嵌入式计算》课程实验指导材料。页面顶端为实验须知、实验平台简介、Bootloader启动流程分析以及3个实验的指导书。页面左侧为指导书的各个小节目录，右侧为小节内的索引，页面右上角可详细搜索，页面下方可切换上下小节。请务必按顺序阅读指导书，有问题积极在群内提出。

如发现指导书编写有误或有宝贵的意见或建议，请私信老师。

若指导书有更新，将第一时间在课程群内推送通知。
